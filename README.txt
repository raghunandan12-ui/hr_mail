# HR Email Automation Project

This project sends personalized emails to HR contacts using an Excel sheet.

## Files

- `send_emails.py` : Python script to send emails.
- `hr_list_updated.xlsx` : Excel file with HR data and job links.

## How to Use

1. Replace `your_email@gmail.com` and `your_app_password` in `send_emails.py` with your actual Gmail and App Password.
2. Install required packages if not already installed:
   pip install pandas openpyxl
3. Run the script using:
   python send_emails.py
